NTC Toledano — Context Window Sweep: CNN+Transformer v1 vs v2
5-fold KFold CV, per-drug regression, H100 80GB HBM3

=== v1 Architecture (baseline) ===
nt embedding → 2x Conv1D → 2x Transformer (full attention) → attention pool → MLP
dropout=0.2, weight_decay=1e-4, batch=512, Huber loss

=== v2 Architecture (improved) ===
(nt + stop-codon-relative pos) embedding → 2x Conv1D
→ 2x Transformer (LOCAL attention, band=7) → stop-codon cross-attn pool → MLP
dropout scaled with context (0.2→0.4), weight_decay=1e-3, batch=512, Huber loss

── KEY TAKEAWAYS ────────────────────────────────────────────────────────────
1. v1 overfits heavily beyond ±10nt; v2 eliminates this with local attention +
   stop-codon query pooling + distance positional encoding + scaled dropout.
2. v2 ±20nt is the best overall: G418 R²=0.624, SRI R²=0.762, DAP R²=0.890.
3. DAP is the most predictable drug (R²=0.89); SJ6986 the hardest (R²=0.55).
4. Next: test v2 on additional context windows (±25, ±30) to find the true
   optimum, and/or run smaller contexts (±6, ±8) with v2.
